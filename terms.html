YCSマッチングプラットフォーム 利用規約

第1条（適用）

本規約は、CTWホールディングス株式会社（以下「当社」といいます）が提供するYCSマッチングプラットフォーム（以下「本サービス」といいます）の利用に関する条件を定めるものです。

利用者は、本規約に同意した上で本サービスを利用するものとします。

本サービスの利用に関しては、本規約のほか、別途定める「プライバシーポリシー」が適用されます。

第2条（定義）

本規約において使用する用語の定義は以下のとおりとします。

  • 「利用者」：本サービスに会員登録を行い、本サービスを利用するYCS会員
  • 「登録情報」：利用者が本サービスに登録した一切の情報
  • 「メンバー」：本サービスに登録している全ての利用者
  • 「公開情報」：本サービス上で他のメンバーに公開される情報
  • 「非公開情報」：当社のみが管理し、他のメンバーには公開されない情報

第3条（利用資格）

本サービスは、当社が運営するYCSの会員資格を有する方のみが利用できます。

第4条（利用登録）

1. 利用希望者は、本規約およびプライバシーポリシーに同意の上、教室内で提供されるプラットフォームに登録することによって利用登録が完了します。

2. 当社は、利用登録の申請者が以下のいずれかに該当する場合、登録を拒否することができます。

  • 過去に本規約違反により登録を抹消された者
  • 登録情報に虚偽、誤記があった場合
  • 第3条の利用資格を満たさない場合
  • 反社会的勢力に該当する、または関係を有する場合
  • その他、当社が登録を不適切と判断した場合

3. 利用登録が完了した時点で、利用者と当社との間で本サービスの利用契約が成立します。

第5条（アカウント管理）

1. 利用者は、登録したメールアドレスとパスワードを自己の責任において厳重に管理するものとします。

2. 利用者は、第三者に自己のアカウントを利用させ、または貸与、譲渡、売買等してはなりません。

3. パスワードは定期的に変更することを推奨します。

4. アカウントの管理不十分、使用上の過誤、第三者の使用等により生じた損害について、当社は一切の責任を負いません。

5. アカウントが第三者に使用されていることが判明した場合、利用者は直ちに当社に連絡し、当社の指示に従うものとします。

第6条（登録情報）

1. 利用者は以下の情報を登録するものとします。

  • お名前（実名または屋号）
  • メールアドレス
  • パスワード
  • 電話番号
  • ビジネス名（法人名または個人事業名）
  • 業種
  • 事業内容
  • 所在地（国・都道府県・市区町村）
  • 提供できる価値・スキル
  • 興味・関心
  • メンバーへのメッセージ
  • 価値観・ミッション（任意）
  • Chatwork ID（任意）
  • SNSアカウント情報（任意）

2. 利用者は、登録情報を常に正確かつ最新の状態に保つものとします。

3. 利用者は、マイページからいつでも登録情報を更新することができます。

4. 登録情報の変更が遅延したことにより生じた損害について、当社は一切の責任を負いません。

第7条（情報の開示範囲）

1. 公開情報（他のメンバーに表示される情報）

  • お名前
  • プロフィール画像
  • ビジネス名
  • 業種
  • 事業内容
  • 所在地（市区町村レベルまで）
  • 提供できる価値・スキル
  • 興味・関心
  • メンバーへのメッセージ
  • 価値観・ミッション
  • Chatwork ID
  • SNSアカウント（登録している場合）

2. 非公開情報（当社のみが管理する情報）

  • メールアドレス
  • パスワード（暗号化して保存）
  • 電話番号
  • 詳細な住所（都道府県・市区町村を超える情報）
  • アクセスログ
  • その他、当社が非公開と判断した情報

3. 当社の管理者は、サービス運営、利用者サポート、規約違反の調査等の必要性に応じて、全ての登録情報にアクセスする権限を有します。

4. 利用者は、前各項の開示範囲に同意した上で本サービスを利用するものとします。

第8条（個人情報の取扱い）

1. 当社は、利用者の個人情報を、別途定める「プライバシーポリシー」に従って適切に取り扱います。

2. 当社は、個人情報保護法その他関連法令を遵守します。

3. プライバシーポリシーは、本サービス上で公開します。

第9条（本サービスの目的と性質）

1. 本サービスは、YCS会員が互いの情報を交換し、交流する場を提供するマッチングプラットフォームです。

2. 本サービスは、メンバー間の情報提供および連絡先交換を支援するものであり、当社はメンバー間で行われる具体的な取引やビジネス活動には関与しません。

3. 利用者は、自己の責任と判断において他のメンバーとの交流およびビジネス活動を行うものとします。

第10条（禁止事項）

利用者は、本サービスの利用にあたり以下の行為を行ってはなりません。

  • 法令または公序良俗に違反する行為
  • 犯罪行為に関連する行為
  • 虚偽の情報を登録する行為
  • 当社、他の利用者または第三者の知的財産権、肖像権、プライバシー権、名誉、その他の権利または利益を侵害する行為
  • 他の利用者に対する嫌がらせ、誹謗中傷、差別的発言等の行為
  • 本サービスのネットワークまたはシステムに過度な負荷をかける行為
  • 本サービスの運営を妨害するおそれのある行為
  • 不正アクセス行為または不正アクセス行為を試みる行為
  • 本サービスの脆弱性を突く行為、またはそれを試みる行為
  • 他の利用者の情報を不正に収集、蓄積、開示または提供する行為
  • 他の利用者になりすます行為
  • 過度な営業活動、宣伝活動、勧誘活動、スパム行為
  • 違法な商品やサービスの販売、宣伝
  • マルチ商法、ねずみ講、またこれらに類する勧誘行為
  • 性的表現、暴力的表現を含む情報を発信する行為
  • 反社会的勢力への利益供与行為
  • 宗教活動または宗教団体への勧誘行為
  • 選挙活動またはこれに類似する行為
  • 他の利用者の本サービス利用を妨害する行為
  • その他、当社が不適切と判断する行為

第11条（利用停止・登録抹消）

1. 当社は、利用者が以下のいずれかに該当する場合、事前通知なく、当該利用者の本サービスの利用を停止し、または登録を抹消することができます。

  • 本規約のいずれかに違反した場合
  • 登録情報に虚偽の事実があることが判明した場合
  • 当社からの連絡に対し、相当期間内に応答がない場合
  • 6ヶ月以上本サービスの利用がない場合
  • 第3条の利用資格を満たさないことが判明した場合
  • その他、当社が本サービスの利用を不適切と判断した場合

2. 前項により本サービスの利用停止または登録抹消となった場合でも、当社は利用者に対して一切の責任を負わず、また利用者に生じた損害の賠償義務を負いません。

3. 当社は、本条に基づき当社が行った措置の理由について、説明義務を負わないものとします。

第12条（退会）

1. 利用者は、当社所定の方法により、いつでも本サービスから退会できます。

2. 退会後も、退会前に発生した利用者の義務は免責されません。

3. 退会後、利用者の登録情報は以下のとおり取り扱います。

  • 個人を特定できる情報：退会後30日以内に削除
  • 統計データ：匿名化した上で保持する場合がある
  • 法令により保存が義務付けられる情報：所定期間保存
  • 他の利用者が保存した公開情報：当社は削除の義務を負わない

4. 退会後の登録情報の復元はできません。

第13条（サービス利用料）

1. 本サービスの利用は、当面の間、無料とします。

2. 当社は、将来的に有料プランを導入する場合があります。その場合、事前に利用者に通知します。有料プランの料金、支払方法等については、別途定めるものとします。

第14条（サービスの変更・中断・終了）

1. 当社は、利用者への事前通知なく、本サービスの内容の変更、追加または廃止ができるものとします。ただし、利用者に重大な影響を及ぼす変更の場合は、可能な限り事前に通知するよう努めます。

2. 当社は、以下のいずれかに該当する場合、利用者への事前通知なく、本サービスの全部または一部を一時的に中断することができます。

  • システムの保守、点検、修理を行う場合
  • コンピューター、通信回線等が事故により停止した場合
  • 火災、停電、天災地変等の不可抗力により本サービスの運営ができなくなった場合
  • 外部サービス（クラウドサービス等）の障害が発生した場合
  • その他、当社が中断を必要と判断した場合

3. 当社は、本条に基づき当社が行ったサービスの変更、中断、終了により利用者に生じた損害について、当社の故意または重過失による場合を除き、一切の責任を負いません。

第15条（免責事項および責任の制限）

1. 当社は、本サービスが利用者の特定の目的に適合すること、期待する機能、商品的価値、正確性、有用性を有すること、利用者による本サービスの利用が利用者に適用のある法令または業界団体の内部規則等に適合すること、継続的に利用できること、および不具合が生じないことについて、明示または黙示を問わず何ら保証するものではありません。

2. 当社は、本サービスに関して利用者が被った損害について、当社の責に帰すべき事由がある場合に限り、以下の範囲で賠償します。

3. 通常損害の範囲内で、かつ直近3ヶ月間に利用者が当社に支払った利用料の総額（無料サービスの場合は金1万円）を上限とする

4. 特別損害、間接損害、逸失利益、データの毀損・消失による損害については賠償しない

5. ただし、当社の故意または重過失による場合は、この限りではない

6. 当社は、以下について一切の責任を負いません。

  • 他の利用者の登録情報の真実性、正確性、信頼性、有用性
  • メンバー間で締結される契約の内容、履行、不履行
  • メンバー間のビジネス取引、連絡、トラブル
  • メンバーの信用性、適格性、資質
  • 本サービスを通じて知り合ったメンバー間で発生した一切の紛争
  • 本サービス外でのメンバー間のやり取りおよびトラブル

7. 本サービスは、他の利用者を紹介、推奨、保証するものではありません。利用者は自己の責任において他の利用者と交流するものとします。

8. 当社は、本サービスに関連して利用者と第三者との間に生じた取引、連絡、紛争等について一切の責任を負いません。

第16条（秘密保持）

1. 利用者は、本サービスを通じて知り得た他の利用者の情報について、当該利用者の同意なく第三者に開示または漏洩してはなりません。

2. 前項の規定は、以下の情報には適用されません。

  • 第7条第1項に定める公開情報
  • 開示時に既に公知であった情報
  • 開示後に利用者の責に帰さない事由により公知となった情報
  • 正当な権限を有する第三者から秘密保持義務を負うことなく適法に取得した情報
  • 法令により開示が義務付けられた情報

3. 本条の規定は、退会後も5年間有効に存続するものとします。

第17条（知的財産権）

1. 本サービスに関する知的財産権は全て当社または当社にライセンスを許諾している者に帰属し、本規約に基づく本サービスの利用許諾は、これらの知的財産権の使用許諾を意味するものではありません。

2. 利用者が本サービスに投稿または登録したコンテンツ（テキスト、画像等）の著作権は、利用者に帰属します。

3. 利用者は当社に対し、投稿コンテンツを本サービスの運営、改善、プロモーション等に必要な範囲で利用（複製、翻案、公衆送信、その他の利用を含みます）する非独占的ライセンスを無償で許諾するものとします。

4. 利用者は、当社および当社から権利を承継しまたは許諾された者に対して著作者人格権を行使しないことに同意します。

第18条（Cookie等の利用）

1. 当社は、本サービスの改善、利用状況の把握、広告配信等のため、Cookie、ウェブビーコン、アクセスログ等の情報を収集します。

2. 利用者は、ブラウザの設定によりCookieを無効化できますが、その場合、本サービスの一部機能が利用できない場合があります。

3. Cookie等の詳細については、プライバシーポリシーをご確認ください。

第19条（通知）

1. 当社から利用者への通知は、登録されたメールアドレス宛の電子メール、または本サービス上での掲示により行います。

2. 電子メールによる通知は、当社が送信した時点で利用者に到達したものとみなします。

3. 利用者が登録したメールアドレスの誤りまたは変更の未届けにより、当社からの通知が届かない場合、当社は一切の責任を負いません。

第20条（規約の変更）

1. 当社は、必要に応じて本規約を変更できるものとします。

2. 規約を変更する場合、当社は以下の方法により利用者に通知します。

  • 重要な変更：変更の効力発生日の30日前までに、登録メールアドレスへの通知および本サービス上での掲示
  • その他の変更：本サービス上での掲示

3. 変更後の規約は、当社が別途定める場合を除き、本サービス上に表示した時点より効力を生じるものとします。

4. 利用者が変更後の規約に同意しない場合、本サービスの利用を中止し、退会することができます。

5. 変更後の規約の効力発生後に利用者が本サービスを利用した場合、当該利用者は変更後の規約に同意したものとみなします。

第21条（連絡・問い合わせ）

本サービスに関する利用者からの当社への連絡・問い合わせは、本サービス上の問い合わせフォームまたは当社が別途指定する方法により行うものとします。

第22条（譲渡禁止）

利用者は、当社の書面による事前の承諾なく、本規約に基づく権利または義務の全部または一部を第三者に譲渡し、または担保に供してはなりません。

第23条（分離可能性）

本規約のいずれかの条項またはその一部が、法令により無効または執行不能と判断された場合であっても、本規約の残りの規定および一部が無効または執行不能と判断された規定の残りの部分は、継続して完全に効力を有するものとします。

第24条（準拠法および管轄裁判所）

1. 本規約の準拠法は日本法とします。

2. 本サービスに関して利用者と当社の間で生じた紛争については、東京簡易裁判所または東京地方裁判所を第一審の専属的合意管轄裁判所とします。

第25条（協議解決）

本規約に定めのない事項または本規約の解釈に疑義が生じた場合は、利用者と当社は誠意をもって協議し、解決を図るものとします。
